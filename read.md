# Yendou-Task

Small TypeScript React app for the Yendou SWE Homework.

## Features

- Counter with `+1` button
- Global counter state via `CounterContext`
- Custom `useCounter()` hook
- Toast notification shown after each increment which is styled to match the Figma spec
- Keyboard focus states,hover feedback and a reset button.

## Run locally

```bash
npm install
npm run dev
```

## Notes

I kept the app small on purpose since I was primarily focused on getting the required pieces correct and I procedeed to spend the rest of the time on toast styling, focus states and making sure the state flow is clean.
